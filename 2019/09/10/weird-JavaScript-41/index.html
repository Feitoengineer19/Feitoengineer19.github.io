JS 簡易捲動（Scroll）效果製作

發表於 2019-09-10 分類於 JavaScript ， Animation

今天上課的時候，老師分享了幾個捲動敘事的網頁作品。

發現自己對 scroll 不是很熟悉，就爬了文章來練習，順手記錄成這篇。

捲動（Scroll）與數位敘事

捲動（Scroll）是網頁中最常見的互動效果，在許多新聞報導的網頁敘事中，都可以看見它的蹤影，配合插圖與圖表，往往能提升讀者的興趣和閱讀體驗。

這裡列舉幾個我看過相當優秀的作品，當作學習的目標：

  • 今晚，我們一起吃塑膠
  • Analysing London 2012
  • A Mission in 4 Mode
  • The Boat
捲動效果製作範例

大部分捲動效果的 JavaScript 邏輯是：當我們將網頁滾動至特定區域（頁面）時，會有相對應的元素移動、改變或顯示／消失。

只要拆解、釐清上面這個邏輯，一定有辦法做出捲動效果，剩下畫面的美感，就得增強設計能力去補足。

今天，我們就透過這個簡單的 DEMO 來練習看看吧！

HTML：元素製作

首先，我們在 HTML 裡面加入我們所需要的元素，包括：

  • 模擬區域頁面 <section>
  • 模擬對應元素 element1 與 element2

最後一個 <section> 加入 class="page3" ，是為了偵測畫面是否進入該區域而特別設定的 class，稍後會詳細說明。

CSS：元素屬性、動畫設定

這裡分別設定兩元素 element1 與 element2 的屬性，主要是透過新增／移除 class 的方式去改變元素的屬性或狀態，搭配 transition 達到過渡的效果。

當然，除了手動設計，我們也可以利用Animate.CSS等套件，來快速產製各式各樣的轉場動畫。

JS：捲動效果的邏輯

這個範例中有兩個捲動效果：

  1. 當畫面捲動超過元素 element1 以後，移動 element1 的位置（Move）。
  2. 當畫面捲動超過頁面 page2 後，顯示 element2 （Show）。

A. 偵測捲動畫面

接著，「偵測捲動畫面」，轉換成 JS 的邏輯是：對整個瀏覽器頁面 window 進行監聽，註冊一個 scroll 事件，當我們捲動畫面時，隨即執行函式 scrollAni。

函式 scrollAni 裡面，我們透過 window.scrollY 偵測瀏覽器頁面的 已經捲動的 Y 軸座標（視窗最頂部），若 Y 軸座標超過特定的值，就改變相對應的元素狀態。

而特定的值，對應到範例中的兩個效果：

  1. 元素 element1 底部的 Y 軸座標，等於元素 element1 距離上層（上方）的 Y 軸座標，加上元素 element1 自身寬度的一半。
  2. 頁面 page2 頂部的 Y 軸座標，等於頁面 page2 距離距離上層（上方）的 Y 軸座標。

透過 offsetTop 讀取元素距離上層（上方）的 Y 軸座標， clientHeight 用來取得元素本身的可視寬度。詳細的指令用法可以參考這篇文章：使用 Javascript 取得元素的座標。
